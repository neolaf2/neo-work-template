# NotebookLM Integration

**Purpose:** Synced NotebookLM notebooks providing AI-powered research and synthesis.

## Setup

```bash
# Link to your NotebookLM repository
ln -s ~/path/to/notebooklm-repo/your-project ./notebooks

# Or clone and link
cd ~ && git clone <your-notebooklm-repo>
ln -s ~/notebooklm-repo/your-project ./notebooks
```

## Usage

NotebookLM provides:
- Automated briefing documents
- Study guides
- FAQs
- Audio overviews

See main project README for integration patterns.

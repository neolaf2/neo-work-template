# Custom Claude Code Skills

**Purpose:** Reusable skills for automating project workflows.

## Categories

- `document-processing/` - Document analysis and extraction
- `content-generation/` - Content creation skills
- `analysis/` - Data analysis and insights
- `utilities/` - Helper utilities

## Usage

Link skills to Claude Code:
```bash
ln -s $(pwd)/skills/document-processing/skill-name \
  ~/.claude/skills/project-skill-name
```

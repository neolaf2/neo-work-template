# Contributing to Agentic Project Template

Thank you for your interest in improving this template!

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest features
- Provide clear description and examples
- Include use case if suggesting new features

### Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test with a real project
5. Commit (`git commit -m 'Add amazing feature'`)
6. Push (`git push origin feature/amazing-feature`)
7. Open a Pull Request

### Adding Examples

We welcome example projects! Add to `examples/` directory:

```
examples/
└── your-example/
    ├── README.md           # What this example demonstrates
    ├── work/               # Sample documents
    └── CLAUDE.md          # Customized for example
```

### Documentation Improvements

Documentation lives in:
- `README.md` - Main overview
- `CLAUDE.md` - Agent instructions
- `doc/` - User guides

## Code of Conduct

Be respectful, constructive, and helpful.
